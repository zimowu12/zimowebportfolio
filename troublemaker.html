Trouble Maker

2+3D turn-based card game made with Unity, in progress

This is a school project that I've been currently working on. It is the final project for graduating. Trouble Maker is a 2+3D game that combines a battle game with card and maze elements. It is a local network game that can be played on 2 devices over the local network. It evaluates Unity's new Netcode for GameObjects package.

Group Members: Cheng Liu, Mukun Liu, Yiru Zhong, Zimo Wu

Game idea

Trouble Maker is a 1v1 competitive turn-based strategy game that takes place in a hedge maze, where one player (the Challenger) needs to find their way out of the maze, while the other player (the Controller) tries to stop him.

Novel idea:

the entire game experience of the Challenger depends on the Controller. Combines a battle game and a maze game with a card game.

Novel technology: Networking

Evaluating Unity’s new Netcode for GameObjects package

My design of card for the game

Target Platform

This game is designed only for Windows operating system under a local network.
